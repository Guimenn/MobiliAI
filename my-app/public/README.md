# Pasta Public

Esta pasta contém arquivos estáticos que são servidos diretamente pelo Next.js.

## Estrutura recomendada:

- `images/` - Imagens do site
- `icons/` - Ícones e favicons
- `documents/` - PDFs e documentos para download
- `fonts/` - Fontes personalizadas

## Como usar:

Os arquivos nesta pasta podem ser acessados diretamente via URL:
- `/favicon.ico` → `public/favicon.ico`
- `/images/logo.png` → `public/images/logo.png`

## Importante:

- Esta pasta DEVE ser commitada no git
- Não adicione arquivos grandes desnecessariamente
- Use nomes de arquivo sem espaços e caracteres especiais
